Fitzo is a modern, responsive landing page designed for a fitness application.

## Features

### Core Sections
- Hero section with CTA buttons
- Workout plans with categories
- Expert trainers showcase
- Pricing plans (Free, Pro, Elite)
- Contact form
- Footer with navigation and branding

### User Profile Page
- Avatar display and upload
- Editable name and email
- Balance display and recharge modal
- Subscription plan display
- Home and logout actions

### Responsive Design
- Fully mobile-friendly layout
- Grid compression for smaller screens
- Adjusted spacing and typography
